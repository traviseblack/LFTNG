Cross-device sync (private GitHub Gist)

Sync your data across devices using your own private GitHub Gist. Set this up once per device: paste a token, then Pull (to load) or Push (to upload). After setup, every saved workout auto-uploads.

How to get a token (one time)

GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic). Give it a name, set Expiration to No expiration, check ONLY the gist scope, generate, and copy it here. It only grants access to gists.

Backup & restore

Your workouts live in this browser. Export a backup regularly (save it in your LFTNG folder). To move devices or recover, import a backup file.
